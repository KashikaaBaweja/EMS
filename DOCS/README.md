## 🧠 The One Rule

> **Give Cursor context + constraints. Vague prompt = vague code.**

❌ `"Make a product page"`
✅ `"Using @app/products/page.tsx as reference, create a [id]/page.tsx that fetches product by ID server-side using Prisma, shows image, title, price, and an Add to Cart button that calls a Zustand store action"`
